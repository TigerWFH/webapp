# 深信服 0 信任管理工具

## aTrust 系统构成

> aTrust 同时支持 HTTPS 代理和 SSL 访问隧道，针对 B/S、C/S 结构应用提供加密隧道

### 控制中心 aTrust-SDPC

> 部署在服务端，核心交换机上

### 安全代理网关 aTrust-Proxy

> 部署在服务端，核心交换机上

### 客户端

> 部署在终端设备上
>
> 在终端设备侧，aTrust 可实现终端安全风险上报及网络数据引流，客户端周期性检测
> 终端环境并上报数据给控制中心，控制中心根据上报结果实时调整访问策略，规避终
> 端出现安全问题时带来的业务风险
>
> 同时，客户端可将指定网络流量打上身份标签、
> 访问`进程信息`到安全代理网关上进行处理，安全代理网关根据访问策略判断是否对流
> 量进行代理。
